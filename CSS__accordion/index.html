CSS Only Accordions Experiment

The following component utilizes hidden radio buttons, combined with their labels and sibling selectors to show/hide the contents of unordered lists.

This demo is merely an experiment to play around with CSS selectors and how they can be used in creative ways. I can not stress enough that this should not be used in production in any way. Use my accordion script instead.
